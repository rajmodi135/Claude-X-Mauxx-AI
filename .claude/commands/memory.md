# Claude x Mauxx AI — Show Memory Index
# Invoke: /memory

Shows the full memory index.

## What It Does
1. Reads memory/MEMORY.md
2. Displays: Active plans, recent archives, rules, facts, state files

## Usage
```
/memory              — show full index
/memory <name>       — load and show a specific memory file
/memory new fact     — create a new fact (interactive prompt)
```
