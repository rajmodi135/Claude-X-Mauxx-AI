# /theme — Switch TUI Theme
# Invoke: /theme [name]

Switch the Mauxx AI TUI color theme.

## Themes

- `neon` (default) — cyan + magenta + green on dark
- `cyberpunk` — yellow + pink + cyan
- `matrix` — green on black
- `vaporwave` — pink + purple + cyan
- `nord` — muted blue + grey
- `monokai` — purple + green + yellow

## Usage

```
/theme           — show current
/theme neon      — set neon theme
/theme matrix    — set matrix theme
```
