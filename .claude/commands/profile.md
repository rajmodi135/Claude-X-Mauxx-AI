# /profile — Switch Profession
# Invoke: /profile [name]

Switch your active profession profile.

## Usage

```
/profile                — show current profile
/profile developer      — software developer
/profile designer       — designer / UI-UX
/profile data-scientist — data scientist / ML
/profile writer         — writer / content
/profile student        — student / learner
/profile devops         — devops / sysadmin
/profile security       — security engineer
/profile manager        — product manager / tech lead
```

Each profile tunes the system:
- Default preset
- Model tier
- Active skills
- Suggestions
- Cost budget
