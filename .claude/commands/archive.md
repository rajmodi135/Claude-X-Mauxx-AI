# Claude x Mauxx AI — Show Past Plans
# Invoke: /archive

Show completed and archived plans.

## What It Does
1. Reads memory/MEMORY.md Recent Archives table
2. Lists all archive-*.md files in memory/
3. For each, shows: completion date, key outcome, lessons learned

## Usage
```
/archive                    — show recent archives
/archive <name>             — show full archive
/archive search <keyword>   — search archives by keyword
/archive stats              — show archive statistics
```
